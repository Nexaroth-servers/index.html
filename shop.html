Intégration Backend

Endpoints recommandés :

  • GET /api/balance?user_id=... → { vote: number, donation: number }
  • POST /api/purchase → { user_id, item_id, method:'vote'|'donation' } → effectue la livraison
  • POST /api/vote-callback → (webhook) créditer points vote à l'utilisateur après vote externe
  • POST /api/donate-callback → (webhook/payment) créditer points donation après paiement

La boutique fournie est statique : il faut un backend sicurisé pour stocker soldes, livrer objets et processeur de paiements.
